# 8. Conclusion

We have presented **V12-Agentic**, a four-agent framework for stock movement onset detection that decomposes the prediction task into Macro Regime Monitoring, Alpha Factor Exploration, Pattern Core classification, and Backtest & Verification. Each agent maps to a concrete component of a deployed quantitative trading system (V12.31, real α +2.236pp/month, Sharpe 2.20 on Chinese A-shares), and we contribute a knowledge encoding methodology that converts the production system's tacit expert knowledge into an LLM-usable prompt.

Through walk-forward random-sample evaluation on 6,000 stock-day anchors, plus an external-validation study on the public ACL18 benchmark, we report four honest findings:

1. **Hybrid LLM-LGBM routing does not robustly beat LGBM** in the realistic deployment distribution. The best hybrid (G_disagreement_boost) ties LGBM's pooled Top-10% return at +1.63%, with overlapping 95% bootstrap confidence intervals.
2. **The apparent stratified-vs-random reversal is within noise, not a base-rate effect.** What looks like a +54% Top-10% advantage in a 25%-positive-rate stratified PoC and a −15% disadvantage under the 8%-positive-rate random sample is, on controlled re-analysis, the difference between two estimates that are each statistically indistinguishable from zero: no composition of the random pool reproduces the advantage (ruling out base rate), a stronger base ranker does not erase it (ruling out baseline weakness), and a date-clustered bootstrap of the headline gap gives +0.72pp with 95% CI [−0.56pp, +2.05pp]. The warning for the field is to report cluster-robust intervals on absolute effect differences rather than ratios.
3. **Cross-quarter heterogeneity creates an oracle ceiling of +0.39pp Top-10% return** for any online regime-conditional router. Our naive routers capture less than 20% of this headroom, suggesting that richer regime conditioning (rolling RankIC tracking, sector momentum embedding, learned regime detectors) is needed to monetize the heterogeneity.
4. **Public stock-movement benchmarks leak temporally into modern LLMs (confirming prior work).** Consistent with an established and active line of work [glasserman2023lookahead, sarkar2024lookahead, lopezlira2023chatgpt, profitmirage2025, ktdfin2026], we confirm — and do not claim as novel — that across the FinBen suite (ACL18 2014–16, BigData22 2019–21, CIKM18 2017) gemini-3.5-flash scores 61–79% directional accuracy with the benchmark context but as high or higher (72–80%) with **only the ticker and date**: removing all input *raises* accuracy on every benchmark, and mega-cap tickers are far easier than obscure ones (up to 95% on CIKM18). The apparent skill is pre-training memorization, not forecasting — a cross-market control nails it down: the same no-context protocol on barely-memorized Chinese A-shares scores 48.6% (chance) versus 72–80% on the US benchmarks. LLM-finance evaluations must use data post-dating the model's training cutoff; we therefore report C1–C4 on a leakage-resistant 2025 walk-forward sample.

These findings constitute a methodological warning for the LLM-finance benchmarking community and a concrete research target for follow-up work.

Pathway 2 of our research program upgrades the Pattern Core to a Temporal Convolutional Network with Spatio-Temporal Cross-Attention. Walk-forward evaluation across the same three splits shows Phase 2 TCN achieves mean Top-10% return +1.61% vs LGBM +1.63% — equivalent performance with **35× less training data**.

Pathway 3 further pretrains the encoder with the Barlow Twins SSL redundancy-reduction objective on 100K unlabeled anchors per split, then finetunes on 30K labeled anchors. The Phase 3 BT-SSL + TCN Pattern Core achieves mean Top-10% +1.63% — **matching Phase 1 LGBM exactly with 116× less labeled data**, and importantly delivers **+95% Top-10% improvement on the LGBM-failure quarter (Split 2 Q3)** (+1.72% vs LGBM +0.88%, RankIC +0.058 vs LGBM −0.015). At deployment, Phase 3 requires no LLM inference cost, making it the strongest practical migration path for the V12-Agentic framework.

The 3-Phase progression — LGBM (3.5M labels) → TCN scaling (100K labels) → BT-SSL pretraining (30K labels) — illustrates a clear path to deployment-grade Pattern Cores under increasingly tight labeled-data budgets. All three phases are scaffolded by the V12-Agentic framework introduced here, ensuring that representation upgrades can be measured against a controlled, deployment-realistic baseline.

We release the full evaluation harness, walk-forward sampler, hybrid router suite, and the V12.31 expert prompt as supplementary material to enable reproduction and extension of these results.
